# 💳 Pricing Card

A modern and responsive pricing section built using **HTML** and **CSS**. This project displays three subscription plans—**Basic**, **Pro**, and **Enterprise**—using Flexbox for layout and hover effects for interactivity.

## ✨ Features

- Three pricing plans (Basic, Pro, Enterprise)
- Responsive layout using Flexbox
- Beautiful purple gradient background
- Highlighted Pro plan with a unique color
- Hover effects on subscription buttons
- Rounded cards with clean spacing
- Responsive pricing layout with `/month` aligned beside the price

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Linear Gradients
- CSS Transitions
