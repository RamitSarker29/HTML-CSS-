# TechFest Registration Form

A simple TechFest registration form built using HTML. This project allows participants to register for different technical and non-technical events.

## Features

- Participant name input
- Email input
- Department selection using a dropdown
- Roll number/Student ID input
- Multiple event selection using checkboxes
- Register button to submit the form

## Technologies Used

- HTML5

## Project Structure

```
TechFest/
│── index.html
```

## Events Available

- Hackathon
- Robotics
- Photography
- Videography
- Debate
- Quiz
- Poster Design
- Cosplay
- Aptitude
- Coding Challenges
