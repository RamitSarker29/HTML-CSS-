# 👟 Product Card

A modern and responsive product card built using **HTML5** and **CSS3**. This project showcases a Converse Chuck Taylor All Star shoe with a clean card layout, gradient background, interactive buttons, and smooth hover effects.

## ✨ Features

- Modern product card design
- Centered layout using Flexbox
- Gradient background
- Rounded image and card corners
- Product image display
- Product title and price section
- "Buy Now" and "Add to Cart" buttons
- Smooth hover animations on buttons
- Responsive and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Gradients
- CSS Transitions
- Media Queries
