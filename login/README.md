# Login Page

A simple and responsive login page built using HTML and CSS.

## Features

- Clean and modern UI
- Centered login card using Flexbox
- Email and password input fields
- Login button with hover effect
- Responsive layout

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
Login_Page/
│── index.html
```

## Preview

The page includes:

- Password input
- Login button
- Hover effects on inputs and button
