# Product Card

A modern product card built using **HTML5** and **CSS3**. This project showcases a clean card layout for an e-commerce product, featuring a product image, pricing, discount badge, and a call-to-action button.

## Features

- Responsive card layout
- Product image with rounded corners
- Product title and description
- Price display with discount badge
- "Buy Now" button with hover effect
- Gradient background
- Card and button shadow effects

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Gradients
- Box Shadow
- Hover Effects
- Border Radius

## Concepts Practiced

- Flexbox for layout
- Card design
- Typography
- Button styling
- CSS gradients
- Box shadows
- Spacing using margin and padding
- Visual hierarchy
