# Coming Soon Landing Page

A modern "Coming Soon" landing page built using **HTML5** and **CSS3**. This project showcases a clean card-based layout with a gradient background, a logo, and an email subscription form for users to receive future updates.

## Features

- Modern gradient background
- Centered card layout using Flexbox
- Custom Google Font
- Logo section
- Email subscription form
- Hover effects on buttons and input fields
- Clean and responsive design

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Google Fonts

## Concepts Practiced

- Flexbox for centering content
- CSS Gradients
- Forms and Input Elements
- Buttons and Hover Effects
- Border Radius
- Typography with Google Fonts
- Responsive Layout Basics
